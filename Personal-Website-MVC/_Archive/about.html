HTML (Hyper Text Markup Language)

The backbone of the web, serving as the standard markup language for creating and structuring web pages. My knowledge in HTML5 ensures that I have all the tools necessary to structure web pages and build functional and attractive websites from scratch.

CSS (Cascading Style Sheets)

The powerful styling language used to enhance the visual presentation and layout of HTML elements on web pages. With CSS3 we can define the appearance and styling of just about anything to create visually appealing, responsive, and consistent designs. Combining my knowldege of HTML with that of CSS truly unlocks the potential to build almost any website you can dream of.

JavaScript (JS)

JavaScript is a dynamic programming language that adds interactivity to web pages. It's versatility in creating interactive elements, handling user input, and manipulating webpage content in real-time makes it such a unique language that can be used on both the client-side and even the server-side (especially through Node.js). Although I am still new to JavaScript, I am able to use it to create dynamic and engaging web experiences for users.
